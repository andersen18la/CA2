A description on how you designed the system

We designed our chatsystem with the jFrame-gui, where we used these attributes: label, text field, text area and buttons. We used this gui as a interface for our client to connect to the server and chat with another client.

how to use the system

First you should start the server (only if the server on our droplet is down). Then run the gui-class: chat form and the interface will pop up. First you have to enter jdbh.dk and port: 8081 in order to connect to the server. Then push the connect button. If you succeeded in getting a connection now you should log in with a username, by putting in your name in the field and clicking the login button. After this you should now be able to send messages. To send messages to all you simply just write a message and press send. However if you would like to send a message to one specific person you need to write the name first then a semicolon and then the message (example: frank:whatsup), if you would like to send to more than one person you write it like this: frank, hans, whatup again.
We've also made it possible to wee which users are online. The text area that shows this (to the right) shows the online users as soon as a client connects with a name.
